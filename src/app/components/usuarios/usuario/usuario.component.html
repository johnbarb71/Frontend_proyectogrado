Usuarios

Menú principal de Usuario

Agregar usuarios

Agregue usuarios de forma fácil, solo llene los datos básicos como nombre, contraseña, correo electrónico, tipo de rol y si es activo.

Actualizar usuarios

Modifique los usuarios de manera ágil, seleccione el usuario dale click en el botón y modifique los parámetros.

Cambiar claves

Restablezca la clave de cualquier usuario.

Sucursales

Puede agregar o eliminar usuarios a sucursales para mayor control.

¡Recuerde!

No puede modificar la cuenta principal.
